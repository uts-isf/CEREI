5.1.3 Part 3: Definition of the Meter-Specific Parameters

This part defines the monthly meter-specific parameters for an entire year (12 months) for each meter. An extract from one single meter is shown in Figure 5.


Figure 5: Part 3 of Input 1 - Network Tariff.


Against each meter number, the NMI number is provided, along with the following parameter information:


1. Loss Ratio - Spot Price (c/kWh, monthly) (Column B)

2. Loss Ratio - Feed-in Tariff (c/kWh, monthly) (Column C)

3. Demand Critical Peak (kVA, monthly) (Column D)

4. Demand Capacity (kVA, monthly) (Column E)


An unlimited number of meters can be listed. Meter information, as shown in Figure 5, simply needs to be replicated below sequentially.

Information relation to all the meters listed in the Input 2 (See Section 5.1) and Input 4 (See Section 5.4) files must be provided in this Input 1 file.
