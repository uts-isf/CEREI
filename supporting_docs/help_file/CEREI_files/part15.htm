5.4 Input 4: Energy Generated‌

The input provides the on-site energy generation data in kWh. The data resolution is 30 minutes. Taking into account the different scenarios in Table 2:

1. Scenario 2 - If the energy generated on site is connected to its own separate meter different from the ones listed in Input 2 (See Section 5.2), then the specific meter name should be provided as the column heading. In this example (as shown in Figure 8), it is called “Virtual NMI”.

2. Scenario 3 - If the generated energy on site is connected to any existing meter listed in Input 2 (See Section 5.2), then the same NMI should be provided as column heading.

3. Scenario 4 - If the generated energy on site is distributed among existing meters listed in Input 2 (See Section 5.2), then the heading of the column becomes irrelevant. This should be clearly stated in Input 1 (See Section

5.1.1 when defining the meter configuration.
